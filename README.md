**Project Overview**

This repository provides a set of lightweight utilities designed for common tasks in educational and development workflows, including parameter handling, configuration management, text comparison, and automated student assignment evaluation.

**Tools**

**1. Internal Toolset `class_aat`**  
A utility class for encapsulating and analyzing variable arguments passed to the main function. It simplifies argument parsing and is particularly useful in command-line tools or scripts where flexible input handling is required.

**2. Configuration File Reader**  
A dedicated toolset for reading and managing configuration files, supporting standard formats and enabling easy integration into larger projects.

**3. Text Comparison Tool `txt_compare`**  
A file comparison utility that identifies differences between two text files. It is suitable for version checking, output validation, and log analysis.

**4. Student Assignment Checker**  
An automated evaluation tool that executes student-submitted executable files (compiled from source code), feeds them predefined inputs, captures outputs, and compares the results against reference answers. It then generates a final score based on the comparison. This tool is especially useful in educational settings for streamlining assignment grading.

**Purpose**

This project aims to reduce repetitive manual work in code testing, assignment evaluation, and file analysis by providing simple, reusable, and easily extensible components. Each tool is designed with clarity and practicality in mind, making it suitable for use in small-scale projects, teaching environments, or as building blocks for larger systems.

**Target Audience**

- Educators and teaching assistants looking to automate student assignment grading.
- Developers seeking lightweight utilities for file comparison, argument parsing, or configuration handling.
- Students interested in understanding basic tool design and automation.

**Status & Contributions**

The tools are currently in stable use and can be extended to support additional features as needed. Contributions and suggestions are welcome.
